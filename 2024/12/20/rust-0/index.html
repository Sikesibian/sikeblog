Rust 安装过程中遇到的一些小问题

  • 不必先安装 Visual Studio，可以直接安装 Rust。建议先安装 MinGW64（ucrt），然后选择 rustup-init 的选项 2 自行配置。点击跳转参考1，点击跳转参考2

  • 建议检查一下 gcc 版本，别太低了。。。（与安全选项有关）

  • 调整安装路径。方法：自行配置环境变量 RUSTUP_HOME 和 CARGO_HOME 点击跳转参考

  • 出现报错 error: linker `link.exe` not found，方法（点击跳转参考）

    依次执行以下命令即可

    rustup toolchain install stable-x86_64-pc-windows-gnu
rustup default stable-x86_64-pc-windows-gnu
