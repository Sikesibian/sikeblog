因子分解算法的思路

因子分解问题（Factoring Problem）

给定正整数 N=∏i=1mpiαiN = \prod\limits_{i = 1}^{m}p_i^{\alpha_i}N=i=1∏m​piαi​​ ，对其进行素因子分解。

经典筛法的核心策略：找两个正整数 z<y<Nz < y < Nz<y<N，让它们满足 z2≡y2(modN)z^2\equiv y^2 \pmod{N}z2≡y2(modN)（平方同余关系）。（当然这里仅考虑 gcd⁡(z,N)=gcd⁡(y,N)=1\gcd(z, N) = \gcd(y, N) = 1gcd(z,N)=gcd(y,N)=1 的情况，否则 NNN 就直接被 zzz，yyy 分解了。）

z2≡y2(modN)  ⟺  (y−z)⋅(y+z)≡0(modN)z^2 \equiv y^2 \pmod{N} \iff (y - z)\cdot (y + z) \equiv 0 \pmod{N} z2≡y2(modN)⟺(y−z)⋅(y+z)≡0(modN)

这意味着只要 y−z>1y - z > 1y−z>1 且 y+z≠Ny + z \neq Ny+z​=N，那么就有 h=gcd⁡(y−z,N)h = \gcd(y - z, N)h=gcd(y−z,N) 为 NNN 的一个非平凡因子。

这时对其做一些简单的变换：

  • 考虑 gcd⁡(y,N)=1\gcd(y, N) = 1gcd(y,N)=1：

z2≡y2(modN)  ⟺  (zy−1)2≡1(modN)z^2 \equiv y^2 \pmod{N} \iff \left(zy^{-1}\right)^2 \equiv 1 \pmod{N} z2≡y2(modN)⟺(zy−1)2≡1(modN)

  • 设 x=zy−1x = zy^{-1}x=zy−1：

x2≡1(modN)x^2 \equiv 1\pmod{N} x2≡1(modN)

(x−1)(x+1)≡0(modN)(x - 1)(x + 1)\equiv 0 \pmod{N} (x−1)(x+1)≡0(modN)

即得到下述 Statement：

Statement 1：对正整数 NNN，方程 x2≡1(modN)x^2\equiv 1\pmod{N}x2≡1(modN) 的解 xxx 满足 2⩽x⩽N−12\leqslant x \leqslant N - 12⩽x⩽N−1，则 gcd⁡(x−1,N)\gcd(x - 1, N)gcd(x−1,N) 和 gcd⁡(x+1,N)\gcd(x + 1, N)gcd(x+1,N) 之中至少有一个是 NNN 的非平凡因子。（x=2x = 2x=2时，x−1=1x-1 = 1x−1=1；x=N−2x = N-2x=N−2时，x+1=N−1x +1=N-1x+1=N−1。）

这时问题就变成了，我们怎么去找到这样的一个关系呢？

  • How to Find?（注：因子222的检测是平凡的，因此下面均考虑 NNN 为奇数。）
  • 欧拉定理？（xφ(N)≡1(modN)x^{\varphi(N)}\equiv 1\pmod{N}xφ(N)≡1(modN)） ⇒\Rightarrow⇒ Nope！ 我们无法计算 φ(N)\varphi(N)φ(N)！
  • 我们可以使用量子傅里叶变换求取元素的阶！（解决元素求阶问题）⇒\Rightarrow⇒ 我们可以计算使得 xr≡1(modN)x^r\equiv 1\pmod{N}xr≡1(modN) 最小的 rrr ！（元素 xxx 在模 NNN 意义下的阶 r=ordN(x)r = \text{ord}_N(x)r=ordN​(x)。）
